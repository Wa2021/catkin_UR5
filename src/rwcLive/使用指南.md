# Robotiq 腕部相机使用指南

## 📦 工具说明

本目录包含 Robotiq 腕部相机的完整工具集，用于视觉伺服开发。

### 🔧 可用工具

| 文件 | 说明 | 使用场景 |
|------|------|----------|
| `rwcLive.py` | GUI 实时查看器 | 快速查看相机画面 |
| `test_robotiq_camera.py` | 相机测试脚本 | 测试连接、保存图像 |
| `robotiq_camera_node.py` | ROS 节点 | ROS 系统集成 |
| `robotiq_camera.launch` | Launch 文件 | 启动 ROS 节点 |

## 🚀 快速开始

### 1. 测试相机连接

```bash
cd /home/xsh/catkin_UR5/src/rwcLive
python3 test_robotiq_camera.py --ip 192.168.0.1
```

✅ 成功会显示：
- 连接状态
- 图像尺寸和信息
- 保存的图像文件名

### 2. GUI 实时查看

```bash
cd /home/xsh/catkin_UR5/src/rwcLive
python3 rwcLive.py
```

🎯 默认 IP 已设置为 `192.168.0.1`，打开即可查看实时画面。

### 3. ROS 集成（推荐用于视觉伺服）

```bash
# 启动相机节点
roslaunch rwcLive robotiq_camera.launch robot_ip:=192.168.0.1

# 在另一个终端查看图像
rosrun image_view image_view image:=/robotiq_camera/image_raw

# 或使用 rqt
rqt_image_view
```

**发布的 Topics：**
- `/robotiq_camera/image_raw` (sensor_msgs/Image) - 原始图像
- `/robotiq_camera/camera_info` (sensor_msgs/CameraInfo) - 相机信息

## 💻 视觉伺服开发

### 完整系统启动

```bash
# 终端 1: 启动机械臂和夹爪
roslaunch ur_planning start_robot_only.launch robot_ip:=192.168.0.1

# 终端 2: 启动相机节点
roslaunch rwcLive robotiq_camera.launch robot_ip:=192.168.0.1

# 终端 3: 运行视觉伺服程序
rosrun your_package visual_servo_node.py
```

### 示例：订阅相机图像

```python
#!/usr/bin/env python3
import rospy
from sensor_msgs.msg import Image
from cv_bridge import CvBridge
import cv2

class VisionServo:
    def __init__(self):
        rospy.init_node('vision_servo')
        self.bridge = CvBridge()
        
        # 订阅相机图像
        self.image_sub = rospy.Subscriber(
            '/robotiq_camera/image_raw',
            Image,
            self.image_callback
        )
    
    def image_callback(self, msg):
        # 转换为 OpenCV 格式
        cv_image = self.bridge.imgmsg_to_cv2(msg, "bgr8")
        
        # 图像处理
        # 1. 目标检测
        # 2. 位置计算
        # 3. 发送运动指令
        
        # 显示图像（调试用）
        cv2.imshow("Camera", cv_image)
        cv2.waitKey(1)
    
    def run(self):
        rospy.spin()

if __name__ == '__main__':
    node = VisionServo()
    node.run()
```

## 🔍 相机 API

Robotiq 腕部相机通过 HTTP 接口提供：

```
GET http://192.168.0.1:4242/current.jpg?type=color
```

**图像参数：**
- 分辨率：640 x 480
- 格式：JPEG
- 刷新率：约 10 Hz

```
│ rwcLive Tools   │ ← 本工具集

│ - ROS Node      │
└────────┬────────┘

         ↓
┌─────────────────┐
│ 你的视觉算法    │ ← OpenCV/YOLO/etc
│ - 目标检测      │
│ - 位置计算      │
└────────┬────────┘
         │ MoveIt Commands
         ↓
┌─────────────────┐
│ UR5 + HandE     │ ← 机械臂 + 夹爪
│ (MoveIt)        │
└─────────────────┘
```
